# Attendance App

An Android-based Attendance Management Application developed to simplify student attendance tracking and class management.

## Features

- Student Attendance Management
- Class Management
- Daily Attendance Reports
- Monthly Attendance Reports
- Summary Reports
- Timetable Management
- Home Screen Widgets
- Data Export Support
- Simple and User-Friendly Interface

## Technologies Used

- Java
- Android Studio
- Room Database & SQL 
- Material Design Components
- Firebase


## Installation

1. Clone the repository

```bash
git clone https://github.com/kachariyaparth2007-tech/Attendance-App.git
```

2. Open the project in Android Studio

3. Register ATTND app in firebase with com.example.attnd package and get google-services.json file and replace it with current google-services.json

4. Sync Gradle and Run
